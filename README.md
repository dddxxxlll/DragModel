# DragModel
一个可拖动可镶嵌的模态框
实现功能：
1. 模态框可拖动
2. 在模态框中打开模态框时，下面的模态框不可点击

使用方法：
1. open_s(id_to, id_from)：第一个参数为要打开的目标模态框id，第二个参数为从哪个模态框打开
2. close_s(id_this, id_to)：第一个参数为要关闭的模态框（当前模态框）id，第二个参数为关闭当前模态框后要前往的模态框
